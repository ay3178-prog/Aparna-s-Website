Run — Arduino Joystick + TouchDesigner Installation

Reimagined the arcade game Run by replacing the keyboard with an Arduino joystick controller. Live sensor data streams into TouchDesigner, generating a real-time 3D visual tunnel that responds dynamically to every movement. A complete physical computing pipeline from hardware wiring and serial communication to generative visual output.
